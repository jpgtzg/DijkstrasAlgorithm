# Dijkstra's Algorithm

This is a simple implementation of Dijkstra's Algorithm in Python.

## Installation

```bash
git clone https://github.com/jpgtzg/Dijkstra-s-Algorithm.git
cd Dijkstra-s-Algorithm
uv sync
```

## Usage

```bash
uv run python main.py
```
