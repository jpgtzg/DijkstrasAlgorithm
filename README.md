# Dijkstra's Algorithm

This is a simple implementation of Dijkstra's Algorithm in Python.

## Installation

```bash
git clone https://github.com/jpgtzg/DijkstrasAlgorithm.git
cd DijkstrasAlgorithm
uv sync
```

## Usage

```bash
uv run python main.py
```
